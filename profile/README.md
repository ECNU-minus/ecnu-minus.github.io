## Hi there 👋

## 我们是谁

我们是华东师范大学一些想做点有趣小东西的学生，
希望可以通过一些怪点子来给大家的学习生活提供些许便利。

ECNU-Minus的名称来自于[SJTU-Plus](https://github.com/SJTU-Plus)，
LOGO尚在绘制中。

我们的口号是「共而饮水，舟而创人」，
即「在学海中与志同道合者共同汲取水分；乘舟渡于学海之上，在旅途中创造出新的人生价值。」
